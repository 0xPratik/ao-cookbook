# コンセプト

aoには多くのコンセプトが組み込まれていますが、コアとなるコンセプトは非常にシンプルな原則に基づいています：

- 2つのコアタイプ: メッセージとプロセス
- 共有状態なし、ホログラフィック状態のみ
- 分散型コンピュータ（グリッド）
