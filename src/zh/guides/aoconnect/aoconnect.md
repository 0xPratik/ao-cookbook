# aoconnect

**aoconnect** 是一个用于同 NodeJS系统或浏览器交互的 Javascript/Typescript 库。
本节中的指南提供了如何利用 **aoconnect** 的代码片段。所有的代码片段均用 Javascript 编写，但应该很容易转换为 Typescript。
